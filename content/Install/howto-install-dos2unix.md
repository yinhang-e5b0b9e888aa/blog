# 在CentOS, Fedora或RHEL上安装

```
yum install dos2unix
```

# 在Ubuntu或Debian上安装

```
apt-get install tofrodos
ln -s /usr/bin/fromdos /usr/bin/dos2unix
ln -s /usr/bin/todos /usr/bin/unix2dos
```
