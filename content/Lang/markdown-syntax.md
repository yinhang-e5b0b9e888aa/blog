
```

```

# URL：直接引用是方括号和圆括号，通过定义引用是两个方括号，最简单用尖括号

```
[直接引用URL](https://yinhang.blog)
```

[直接引用URL](https://yinhang.blog)

```
[通过定义引用URL][URL定义]
[URL定义]: https://yinhang.blog
```

[通过定义引用URL][URL定义]
[URL定义]: https://yinhang.blog

```
[通过定义引用URL][]
[通过定义引用URL]: https://yinhang.blog
```

[通过定义引用URL][]
[通过定义引用URL]: https://yinhang.blog

```
<https://yinhang.blog>
```

<https://yinhang.blog>
